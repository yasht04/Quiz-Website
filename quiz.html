About QuizMaster

QuizMaster is a dynamic quiz application built to make learning and testing knowledge an engaging experience. We use the Open Trivia Database to provide a vast range of questions across numerous categories and difficulties.

This project was created using HTML, Tailwind CSS, and vanilla JavaScript, demonstrating a lightweight yet powerful single-page application.
